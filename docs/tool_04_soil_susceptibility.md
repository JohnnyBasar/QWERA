# Tool 4 – Soil Susceptibility to Wind Erosion

## Description
Combines two classified rasters:
- Soil erodibility raster
- Wind protection raster

Using the WERA combination matrix (Funk & Völker, 2024), each pixel pair is mapped to a final susceptibility class.

## Parameters
- Soil erodibility raster
- Wind protection raster
- Output raster
- Optional auto-load

## Output
- Susceptibility raster (single-band, class-coded)
